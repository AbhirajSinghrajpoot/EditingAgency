Welcome to the repository of **Editkaro.in**, a modern, conversion-focused landing page and portfolio website designed for a video editing agency. 

This website serves as a digital storefront to showcase high-quality video editing services, ranging from fast-paced Reels and gaming montages to commercial ads and cinematic anime edits.

🔗 **Live Demo:** [editing-agency-beryl.vercel.app](https://editing-agency-beryl.vercel.app/)

---

## 🌟 Key Features

* **Premium Dark UI:** A cinematic and modern dark theme that makes video and image thumbnails pop.
* **Dynamic Portfolio Gallery:** An interactive portfolio section with category filters (All, Reels, Ads, Gaming, Anime) powered by Vanilla JavaScript.
* **Responsive Design:** Fully optimized for all devices (Desktop, Tablet, and Mobile) ensuring a seamless browsing experience.
* **Conversion-Optimized CTA:** Strategically placed "View Portfolio" and "Get a Quote" buttons to drive client engagement.
* **Multi-Page Architecture:** Clean routing between Home, Portfolio, About, and Contact sections for organized content delivery.

## 📂 Project Structure

The project is built using pure vanilla web technologies. Here is the repository structure:

```text
📦 FullEditkaro
 ┣ 📂 assets/            # Images, icons, and media files used in the UI
 ┣ 📜 index.html         # Main landing page (Hero section & overview)
 ┣ 📜 portfolio.html     # Interactive gallery showcasing editing work
 ┣ 📜 about.html         # Agency mission and detailed service list
 ┣ 📜 contact.html       # Client inquiry and contact forms
 ┣ 📜 style.css          # Global styling, layout, and animations
 ┣ 📜 script.js          # Logic for portfolio filtering and interactive elements
 ┗ 📜 README.md          # Project documentation

```
## 🚀 How to Run Locally
Since this is a static frontend project, setting it up locally is incredibly simple:
 1. **Clone the repository:**
   ```bash
   git clone [https://github.com/AbhirajSinghrajpoot/FullEditkaro.git](https://github.com/AbhirajSinghrajpoot/FullEditkaro.git)
   
   ```
 2. **Navigate to the directory:**
   ```bash
   cd FullEditkaro
   
   ```
 3. **Launch the project:**
   * Simply double-click the index.html file to open it in your default web browser.
   * *Alternatively*, if you are using VS Code, use the **Live Server** extension to open index.html for hot-reloading as you make changes.
